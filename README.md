# Chong Xian — Personal Portfolio

## Local development
1. Copy `.env.local` from `.env.example` and fill secrets.
2. Install: `npm install`
3. Run dev: `npm run dev`
4. Build: `npm run build`
5. Start production locally: `npm run start`

## Repo structure
- `assets/` — images, resume, 3D candidate files
- `content/` — YAML content for projects/certifications/education
- `design/` — design tokens
- `pages/` — Next.js pages
- `components/` — React components
- `codex_state.json` — progress & checkpoint file for Codex
