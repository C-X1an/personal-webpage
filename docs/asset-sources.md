# Asset Sources

## 3D assets used in Task 03/04

No external GLB assets were downloaded for this proof-of-concept.

The desktop garden scene currently uses procedural low-poly geometry built in
code inside `components/GardenCanvas.jsx` for:

- sakura tree placeholder
- fountain placeholder
- playground pavilion placeholder
- contact kiosk placeholder
- path stones and shrubs

This keeps the first interactive reveal lightweight while avoiding a blank
canvas during the hero-to-garden transition.
